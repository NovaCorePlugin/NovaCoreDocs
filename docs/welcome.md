# NovaCore Documentation

Welcome to the official documentation of the **NovaCore ecosystem**.

NovaCore is a modular suite of Minecraft server plugins designed to power advanced gameplay systems such as quests, jobs, economy, influence and more.

Instead of relying on isolated plugins, NovaCore provides a connected ecosystem where multiple modules can work together to create deeper and more dynamic server experiences.

## Why NovaCore?

NovaCore plugins are designed with three main goals in mind:

- **Modularity** – each plugin works independently but can integrate with other NovaCore modules.
- **Configurability** – server owners have full control over behavior through simple configuration files.
- **Scalability** – plugins are built to support both small community servers and large network environments.

## Available Modules

Currently available plugins in the NovaCore ecosystem include:

- **NovaWild** – Controlled wilderness teleportation with zone-based spawning.
- **NovaQuest** – A configurable quest system designed for structured server progression.

Additional modules are currently in development and will be documented as they are released.

Each plugin is designed to extend the ecosystem and interact with other modules to create a cohesive server experience.
