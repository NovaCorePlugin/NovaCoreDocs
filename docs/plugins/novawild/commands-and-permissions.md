# Commands & Permissions

### Commands

#### `/novawild tp`

Shows the available teleport zones in chat.

#### `/novawild tp <zone>`

Teleports the player to the selected zone.

Example:

`/novawild tp north`

#### `/novawild reload`

Reloads the plugin configuration.

### Permissions

At the moment, permissions are not yet documented in detail.

Recommended future structure:

* `novawild.use`
* `novawild.admin`

### Notes

The `/novawild open` command is planned for future GUI support and is not part of the public release documentation yet.
