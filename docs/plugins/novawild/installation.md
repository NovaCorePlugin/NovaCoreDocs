# Installation

## Installation

### Requirements

* Paper / Spigot 1.20+
* Java 17+

### Installation Steps

1. Download the NovaWild jar file.
2. Place it inside your server `plugins` folder.
3. Restart the server.
4. Open the generated plugin folder.
5. Configure your zones and map boundaries inside the config file.
6. Reload or restart the server.

### Notes

NovaWild has no external dependencies and works as a standalone plugin.

After installation, make sure that:

* each zone has valid center coordinates
* map min/max boundaries are correct
* your ranges do not exceed the playable map
