# Introduction

<figure><img src=".gitbook/assets/novacore.png" alt=""><figcaption></figcaption></figure>

NovaCore is a modular ecosystem of plugins designed to power advanced Minecraft servers.

Plugins included in the ecosystem:

* NovaQuest
* NovaInfluence
* NovaProfession
* NovaWild
* NovaClimate
* NovaJob
* NovaGui
